## 这是啥？
这是一个快速登录小米路由器的脚本（我无法接受小米路由器WEB的页面卡成一坨狗屎，等待加载时间实在太长）
## 能做什么？
一键登录、断开重新拨号（用于更换IP）、重启路由器
## 注意事项
脚本依赖 Crypto 库，用于生成加密字符串，Python原生没有自带，需要自行安装
